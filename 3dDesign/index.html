is definitly worth it the model is fulley textured with basic colors for different parts including the full interior. some pieces have

another major feature was the cannons on the wings as they have been designed to be able to be easily oriented realistically as shown in
the video using a parent/child strucure

this and the next few sections are about geometry nodes which is a system that allows modular creating of objects rather than hand making


the first project shown is a procedural fence which takes a curve and projects it onto a plane to create a easily movable fence, it also has
editable settings like post distance, amount of horizontal slats and many more

the second example is a thread/string generator which twists threads around a curve and can also be editied with various setting such as a
fray point, as shown in the fifth image
thirdly is a plane with vegetation and rocks which automatically adapt to a river which is attatched to a curve

the procedural rock generaetor is eawsily the most technically difficult project i have done as the amount of geometry node dealings was

is then extruded upwards, this might sound simple in writing but is very complex in execution
